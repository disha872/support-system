# 💬 Smart Customer Support System

A full-stack AI-powered customer support platform with chatbot, ticket management, admin dashboard, and analytics.

---

## 🚀 Features

- 🤖 AI Chatbot (TF-IDF + Cosine Similarity)
- 🎫 Ticket Management System
- 👨‍💼 Admin Dashboard (View + Reply + Resolve)
- 👤 User Authentication (Login/Register)
- 🔐 Role-Based Access (Admin vs User)
- 💬 Chat History Tracking
- 📊 Analytics Dashboard
- 🌐 Full Stack (Frontend + Backend + Database)

---

## 🏗️ Tech Stack

### 🔹 Frontend
- Streamlit

### 🔹 Backend
- FastAPI

### 🔹 Database
- MySQL

### 🔹 AI / NLP
- Scikit-learn (TF-IDF, Cosine Similarity)
